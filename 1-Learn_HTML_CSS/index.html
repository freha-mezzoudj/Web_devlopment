HTML basics:

Block Elements

Block elements: start on a new line and take full width available: div,h1-6, p, form.

Inline Elements

Inline elements: do not start on a new line and take only the necessary width: span, img, a for the link.

Html semantic Tags

A semantic elements describes its meaning to both the browser and the developer. such as : header, footer, aside, main, article, nav, section, details
